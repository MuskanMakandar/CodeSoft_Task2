Web Development

HTML

HTML (HyperText Markup Language) is the standard markup language used for creating web pages. It defines the structure and content of a webpage.

CSS

CSS (Cascading Style Sheets) is used for styling the presentation of a webpage. It defines the layout, colors, fonts, and other visual aspects.

JavaScript

JavaScript is a programming language that enables interactive elements and dynamic functionality on a webpage. It can be used to manipulate the DOM (Document Object Model), handle events, and more.
